Розділ 1: Вступ

Сьогодні ми поговоримо про основи веб-розробки. Веб-розробка охоплює створення та підтримку веб-сайтів, і включає як графічний дизайн, так і програмування.

Знання HTML, CSS та JavaScript є важливими для кожного веб-розробника, оскільки вони становлять основу веб-додатків.

Розділ 2: Основи HTML

HTML (HyperText Markup Language) - це стандартна мова розмітки для створення веб-сторінок. Він використовує теги для визначення структури контенту.

Наприклад, теги <h1> до <h6> використовуються для заголовків, а тег <p> - для абзаців тексту.

Розділ 3: Основи CSS

CSS (Cascading Style Sheets) - це мова стилів, яка використовується для опису вигляду HTML-документів. Вона дозволяє контролювати кольори, шрифти, відступи та багато інших аспектів дизайну.

CSS може бути вбудованим у HTML-документ або зберігатися в окремому файлі.
